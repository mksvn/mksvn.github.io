UX/UI Designer, Front-End Developer

Kiosk ordering system tailored for buffets

Client
Aurora Data Logic is a software engineering company that specializes in custom point of sales systems.

My Role
Design a kiosk ordering system interface that's centered around a casino buffet host's payment process and optimized for tablets.

  • As a buffet host, I want to add guests to the bill as either adults, seniors or children, so they are charged the appropriate amounts.
  • As a buffet host, I want to review an order before payment, so that I can make adjustments if needed.
  • As a buffet host, I want to add a player's card and accept card payments, so I can apply discounts and close the check.
